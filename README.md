# Telegram Finance Bot

## Описание проекта

Telegram Finance Bot — это бот для Telegram, предназначенный для автоматизации финансовых операций и предоставления аналитических данных пользователям.

## Установка и запуск

Для установки и запуска бота выполните следующие шаги:

1. Установите необходимые зависимости:
```bash
pip install -r requirements.txt
```
2. Настройте окружение, следуя инструкциям в файле `example.env`.
3. Запустите бота:
```bash
python main.py
```

## Структура проекта

Проект организован следующим образом:

- `src/`: содержит основной код бота
- `db/`: содержит файлы базы данных
- `docs/`: содержит документацию
- `tests/`: содержит тесты

## Использование

Примеры использования бота:

```bash
/start - запускает бота
/help - отображает помощь
```

## Технические детали

Бот использует:
- База данных: SQLite
- Контейнеры Docker: для упрощения развертывания
- Обработка ошибок: обработка исключений и логирование
